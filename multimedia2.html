EL MUNDO DEL DISEÑO DE SITIOS WEB

 

Un sitio web reune información a través de un conjunto de páginas web relacionadas o categorizadas mediante menús o vínculos enlazados.

Su escritura es a base de código web que se registra mediante editores que soporten html. Dicho código tiene una estructura principal como un encabezado, un cuerpo o contenido y un footer. Un sitio web puede albergar datos tanto de tipo informativo, a través de multimedia como videos, imagenes, audio, etc, como de tipo contable o administración de datos a nivel interno como en las bases de datos, sistemas de gestión financiera, entre otros.

Luego de tener creado el sitio se debe anclar a la web por medio de un hosting y un dominio para ser visible en internet.

A continuación se hace una muestra de la utilidad y proceso de diseño de un sitio web.
